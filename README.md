# ku-yifan.github.io

Personal academic website of **Yifan Sun** — Assistant Professor, Institute for Integrated
Radiation and Nuclear Science (KURNS), Kyoto University.

Live at **<https://ku-yifan.github.io>**.

Built with [al-folio](https://github.com/alshedivat/al-folio) (Jekyll), deployed automatically
to GitHub Pages via GitHub Actions on every push to `main`.

## Updating

- **Publications** — edit `_bibliography/papers.bib` (one BibTeX entry per paper; entries with
  `selected = {true}` also appear on the homepage).
- **Bio / homepage** — edit `_pages/about.md`.
- **Site metadata** — edit `_config.yml` and `_data/socials.yml`.

Push to `main` and the site rebuilds itself (~3 min).
